# FAUCETTA_AGUSTIN_DAW

Se renombraron archivos para simplificar la navegación:

- `clima.html` → `index.html` (archivo principal de la página)
- `clima.css` → `style.css` (hoja de estilos)
- `clima.js` permanece como `clima.js` (lógica JavaScript)

`index.html` enlaza a `style.css` y a `clima.js`.

Para ver la página localmente, abrí `index.html` en un navegador.
